## Design Use Notes

- The front page uses a live CSS/HTML orbit rather than a static screenshot.
- Real ARC SE visuals are used selectively so the site stays premium first and evidence-driven second.
- Dark runtime visuals are framed inside light institutional sections to avoid turning the site into a trading dashboard.
- Every major visual in the website is clickable and enlarges in an in-page lightbox.
- The expanded narrative now treats 3004 as body, 3001 as mind, and 3002 as soul of the core ARC SE organism.
