# ARC SE Video V2 - Style Reference

Reference searched:

- Envato Elements SaaS video templates
- Best-fit inspiration: `Modern Fluid SaaS Product Promo with Dynamic UI Elements`

Source idea, not copied template:

- soft morphing gradients
- dynamic text reveals
- sleek UI panels
- product-screen motion
- clean SaaS product-demo rhythm
- no crypto styling
- no generic stock footage

ARC SE adaptation:

- use the existing website palette: ink navy, blue, blue-deep, violet, teal, gold, mist white
- use the website typography: Georgia-style documentary headlines, Segoe/Inter-style UI text
- use real ARC SE screenshots as the product surface
- animate screenshots with slow zooms, parallax, scan lines, card movement, proof grids, and evidence reveals
- use Watcher, Coordinator, May15, Micro Harvest, Economic Validation, and Evidence Vault as the narrative reveal sequence
- keep caveats visible: evidence, not guarantees

Chosen direction:

Premium technology documentary plus SaaS product reveal. The video should feel like the website came alive, not like a separate motion template.
